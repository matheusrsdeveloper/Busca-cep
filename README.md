# 📍 Busca CEP

Aplicação desenvolvida para praticar o consumo de APIs utilizando **JavaScript**, realizando consultas de CEP através da API pública **ViaCEP**.

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- ViaCEP API

---

## ✨ Funcionalidades

- 🔎 Consulta de CEP em tempo real
- 📍 Exibição de rua, bairro, cidade e estado
- ⚠️ Validação para campo vazio
- ❌ Tratamento de CEP inexistente
- 🌐 Consumo de API utilizando `fetch`
- 💻 Interface simples e intuitiva

---

## ▶️ Como executar o projeto

1. Clone este repositório

```bash
git clone https://github.com/matheusrsdeveloper/Busca-cep.git
```

2. Acesse a pasta do projeto

```bash
cd busca-cep
```

3. Abra o arquivo `index.html` no navegador.

## 📚 O que foi praticado

Durante o desenvolvimento deste projeto foram aplicados conceitos como:

- Manipulação do DOM
- Eventos em JavaScript
- Template Strings
- Funções assíncronas (`async/await`)
- Requisições HTTP com Fetch API
- Consumo de APIs REST
- Tratamento de erros com `try/catch`
- Validação de dados

---

## 🌐 API utilizada

O projeto utiliza a API pública do **ViaCEP** para consulta de endereços brasileiros.

> https://viacep.com.br/
